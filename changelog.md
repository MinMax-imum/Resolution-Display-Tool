
# Changelog - 更新日志

## 版本 0.2.1 (2025-11-08)

### 新增功能
- 程序启动默认选中 1920×1080 分辨率
- 程序启动时输出调试信息到控制台，包括 DPI、缩放比例、全屏分辨率等

### 功能优化
- 减小分辨率列表框的列表项间隔、调整列表框宽度和高度，使显示更紧凑且内容完整展示
- 分辨率列表框列表标题改为“分辨率列表”，去掉冒号
- 移除分辨率列表框的箭头指示，仅保留颜色高亮
- 限制程序帧率为 60 FPS，降低 CPU 占用
- 操作逻辑改为上下方向键切换分辨率以匹配列表顺序
- 左上角显示模式下，分辨率区域从屏幕最左上角绘制，移除 10 像素的边距
- 化信息展示方面优分辨率和缩放比例文字的格式，去掉多余冒号，同时显示宽维度和高维度的缩放比例
- 简化显示模式提示文字，移除重复的操作提示

### 代码优化
- 添加 AI 使用声明 (程序由豆包 AI 生成，经人工完善)
- 关键功能点补充注释

## 版本 0.2.0 (2025-11-07)

### 新增功能
- 扩展分辨率列表：新增 1152×864、1440×1050 等分辨率，总计 18 种常见分辨率
- 左侧增加分辨率列表框，展示所有分辨率及对应长宽比
- 列表框中高亮当前选中分辨率，并用箭头指示
- 显示当前分辨率的长宽比、相对全屏的缩放比例以及放大至全屏所需缩放比例

### 功能修改
- 优化高 DPI 适配逻辑，所有列表元素尺寸和位置均按缩放比例动态调整

## 版本 0.1.2 (2025-11-07)

### 问题修复
- 解决系统缩放比例 (如 125%) 会导致 Pygame 获取屏幕尺寸不符，绘制内容超出屏幕的问题

### 功能修改
- 通过 Windows API (ctypes 调用 user32.dll) 获取系统真实 DPI 和物理分辨率，计算缩放比例
- 所有绘制元素 (字体大小、边距、文字位置等) 乘以缩放比例，适配系统缩放
- 代码仅在 Windows 系统可用，不再支持跨平台

## 版本 0.1.1 (2025-11-07)

### 新增功能
- 增加居中和左上角显示模式切换功能，用空格键触发
- 新增显示模式提示文字，明确当前显示模式

## 版本 0.1.0 (2025-11-07)

- 支持左右方向键切换分辨率和 `ESC` 键退出程序
- 屏幕中央以白色边框标示当前选中分辨率区域
